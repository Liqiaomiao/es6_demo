symbol的声明 Symbol()
Symbol在对象中的应用:构建对象的key,调用key和赋值
Symbol对象元素的保护作用：在对象中有很多值，但是循环输出时，并不希望全部输出，那我们就可以使用Symbol进行保护。
//没有进行保护的写法：
var obj={name:'jspang',skill:'web',age:18};

for (let item in obj){
    console.log(obj[item]);
}

 //现在我不想别人知道我的年龄，这时候我就可以使用Symbol来进行循环保护。
let obj={name:'jspang',skill:'web'};
let age=Symbol();
obj[age]=18;
for (let item in obj){
    console.log(obj[item]);
}
console.log(obj);
